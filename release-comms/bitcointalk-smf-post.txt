[hr]

[size=12pt][b]Why this fork[/b][/size]

The OFF (Offerings to Cthulhu) sibling chain was 51%-attacked into oblivion in 2018. (BOB) currently runs on ~135 MH/s with one or two hash sources — the same NiceHash-spot-rental that killed OFF can target us any week. v0.11.0 hardens (BOB) against that exact attack:

[list]
[li][b]LWMA-3 difficulty adjustment[/b] (Zawy's Linear Weighted Moving Average, N=60, T=120s). Replaces DigiShield V4. Reacts to hashrate swings in ~30 blocks instead of dragging for hours, so a hash-bomb can't ride a stale low-difficulty wave.[/li]
[li][b]100-block max-reorg-depth[/b] (~3.3 hours of finality). Any chain that tries to reorganize past 100 buried blocks is rejected at consensus — full stop. Rented-hash attacks now have to outrun [i]live[/i] consensus difficulty, not slip in via a deep secret-mined replacement.[/li]
[/list]

Combined with the wBOB bridge's 288-confirmation deposit depth (~9.6 hours), money on the bridge is now strictly safer than the consensus finality window. Bridge ↔ chain order is correct.

[size=12pt][b]How to upgrade — Windows[/b][/size]

[list=1]
[li][b]Back up wallet.dat[/b]. Always back up wallet.dat. (BOB) is NOT for Sale, but it is yours to lose.
Default location: [tt]%APPDATA%\Dobbscoin\wallet.dat[/tt][/li]
[li][b]Close[/b] your running Dobbscoin Core wallet completely (File → Exit, then check Task Manager for [tt]dobbscoind.exe[/tt] / [tt]dobbscoin-qt.exe[/tt] — kill any stragglers).[/li]
[li][b]Download[/b] [tt]dobbscoin-0.11.0-win64.zip[/tt] from the link above.[/li]
[li][b]Verify[/b] the SHA256 if you can ([tt]CertUtil -hashfile dobbscoin-0.11.0-win64.zip SHA256[/tt] in cmd) — should match the hash listed above.[/li]
[li][b]Extract[/b] the zip. You'll get [tt]dobbscoin-qt.exe[/tt], [tt]dobbscoind.exe[/tt], [tt]dobbscoin-cli.exe[/tt], [tt]dobbscoin-tx.exe[/tt].[/li]
[li][b]Replace[/b] your old binaries with the new ones (typically in [tt]C:\Program Files\Dobbscoin\[/tt] or wherever you installed the old version — or just run [tt]dobbscoin-qt.exe[/tt] from the extracted folder).[/li]
[li][b]Launch[/b] [tt]dobbscoin-qt.exe[/tt]. Confirm the title bar / Help → About reads [b]v0.11.0[/b].[/li]
[li]Let it sync. Your wallet.dat is untouched — same balance, same addresses, same Slack.[/li]
[/list]
